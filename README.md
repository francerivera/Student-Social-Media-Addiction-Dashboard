Project Overview
This Business Intelligence project explores the relationship between social media usage and the mental well-being of university students. The dashboard provides a comprehensive look at how different platforms affect academic performance and daily life, categorized by student demographics.

Key Features
Impact Tracking: Visualizes the correlation between "Addiction Scores" and "Mental Health Scores" using interactive bubble charts.

Usage Metrics: Monitors daily habits, including average daily usage hours and sleep patterns.

Participant Profiling: Breaks down data by gender, age, relationship status, and year level to identify high-risk groups.

Geographic Distribution: Includes a global map view of participant locations.

Dashboard Pages
Impact Page: Focuses on the "Addicted Score" vs. "Mental Health Score" and identifies the most used platforms like Instagram and TikTok.

Profile Page: Displays student demographics such as year level (with a high concentration of 3rd-year students) and academic performance impact.

Technical Details
Tool: Power BI

Key Metrics:

Daily Usage: Average time spent on social media.

Addicted Score: A calculated metric based on user behavior.

Academic Impact: Percentage of students who feel their performance is affected.

How to Use
Open the .pbix file in Power BI Desktop.

Use the navigation tabs at the top (Home, Impact, Profile) to switch views.

Interact with the slicers to filter data by specific demographics or platforms.
